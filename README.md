# 👤 Random User Generator

A modern and responsive **Random User Generator** built using **HTML, CSS, and JavaScript**. This project fetches random user data from the **Random User API**, allows users to save their favorite profiles using **Local Storage**, and supports **Dark Mode** with theme persistence.

---

## Features

- Generate random users
- Fetch live user data using Random User API
- Save favorite users
- Local Storage support
- Remove favorite users
- Clear all favorites
- Dark / Light mode
- Fully responsive design
- Smooth animations
- Modern UI
